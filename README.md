基于Github Page博客框架（于2019/7月开源）
======================

完全基于前端，不包括任何后台
-------------------------

本套源码主要用到以下几个框架和相关技术：
  * 使用百度统计来记录和分析用户访问的流量
  * 使用畅言来实现用户留言功能
  * 域名使用腾讯云，使用七牛进行访问加速
  * 视频微处理，以实现在线播放
  * 使用HTML5、Canvas等完成博客主体的搭建

* 博客展示效果：https://xiaohegithub.cn/
* 如何使用这套框架：https://www.zhihu.com/question/59088760/answer/161640592
